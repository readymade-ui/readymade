In this example, a class named MyButtonComponent includes a template and styling for a button. MyButtonComponent has methods for handling click when the component emits a 'bang' event.

import { ButtonComponent, Component, Emitter, Listen } from '@readymade/core'; @Component({ selector: 'my-button', template:` {{model}} `, style:` :host { background: rgba(24, 24, 24, 1); cursor: pointer; color: white; font-weight: 400; } `, }) class MyButtonComponent extends ButtonComponent { constructor() { super(); this.state.model = 'Click'; } @Emitter('bang', { bubbles: true, composed: true }) @Listen('click') public onClick(event) { this.emitter.broadcast('bang'); } @Listen('keyup') public onKeyUp(event) { if (event.key === 'Enter') { this.emitter.broadcast('bang'); } } } customElements.define('my-button', MyButtonComponent, { extends: 'button'}); export { MyButtonComponent };

The name 'readymade' is an homage to Marcel Duchamp, an artist in the 20th century who made art with ordinary objects. Like Duchamp's readymades, this library picks up ordinary objects found in ECMAScript, repositions and joins them together to form something completely new: a set of tools that make developing Web Components easier.
